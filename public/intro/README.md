# Cinematic intro video assets

Drop Veo/Sora clips here as they come out of Flow. The scroll container
at `/intro-preview` will pick them up automatically.

Expected filenames (all 16:9, muted, .mp4):

- `beat-0.mp4` — Hero (sunrise horizon)
- `beat-1.mp4` — Human connection (coach + athlete, golden hour)
- `beat-2.mp4` — Invisible coach (translucent figures beside clients)
- `beat-3.mp4` — Community (rooftop circle drone pull-up)
- `beat-4.mp4` — Horizon arc CTA (sky tilt-up)

If a file is missing the section falls back to a black background so
the scroll layout still works during iteration.
